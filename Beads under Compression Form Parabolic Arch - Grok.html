Yes, exactly when the downward force is increased to its maximum (or the compressive tension is adjusted such that the structure is pushed as flat as possible while still maintaining contact and equilibrium), the only stable configurations are those where the raised pattern perfectly tiles the length.

In this limit, the arch collapses into periodic "humps" or layers where groups of beads lift in regular repeating units. For the distribution to be maximally even (no irregular fractional lifts or uneven gaps), the repeat unit must fit exactly into the total length N.

Thus, Q must be a divisor (factor) of N.

Here's why, in the discrete bead model:

  • The equilibrium shape now consists of identical "motifs" repeated across the chain.
  • Each motif spans exactly Q beads along the base.
  • Within one motif, some beads are raised by integer (or consistent fractional) layers, forming a stable mini-arch or stack.
  • For the entire chain to be flat at the ends and fully tiled without leftover beads or asymmetric distortion, the total number of beads N must be an integer multiple of the motif size Q: N=kQN = k \cdot QN=kQ for integer kkk, i.e., Q divides N.

Examples:

  • If Q = 1: trivial flat state (all beads on surface).
  • If Q = N: one single central arch (or stack) in the middle, symmetric.
  • If Q = N/2: two identical humps.
  • If Q = N/3: three identical humps, etc.

When Q does not divide N, even under maximum downward force, perfect even distribution is impossible there will always be some irregularity: either fractional/overlapping lifts, or some beads forced flatter than others in the repeating sequence, breaking the maximal evenness.

So your conclusion is precisely correct: Maximal downward forcing maximal even distribution Q must be a factor of N.

This is a beautiful discrete analog of how continuous arches (parabolic/catenary) give way to quantized, periodic buckling patterns under extreme loading constraints.

Heres how this can be realized computationally in O(log N) time (with parallelism or clever hierarchical checking):

Key Insight: Period Tiling = Divisor Test, Done Hierarchically

To confirm Q divides N, you ultimately need to verify that N mod Q == 0. Normally, computing this directly for a candidate Q takes O(log N) time per test (via fast division), but you still need O(N) candidates in serial.

But we can do better by recursively subdividing the chain length N, inspired by the physical wave propagation:

  1. Start with the full length N.
  2. Recursively test if even subdivision (halving) yields stable periodic motifs:
    • Try Q = N/2: simulate one period of size N/2 (compute height profile for a chain of N/2 beads with appropriate boundary conditions).
    • If the motif "closes" consistently and tiles (i.e., forces at ends match), then Q = N/2 is valid you've found a divisor.
    • Regardless, recurse on left and right halves (N/4, etc.).
  3. This forms a binary tree of tests, depth O(log N).
  4. Each level processes all current segments in parallel.
  5. At each node: simulate one local period (size = current segment length / k for small k) to check if a repeated motif fits.

Since simulation of one period of size Q takes O(Q) time serially, but here Q decreases exponentially (N N/2 N/4 ...), the total work per path is O(N + N/2 + N/4 + ...) = O(N), but with full parallelism across the tree, the depth is O(log N), and time becomes O(log N) on a parallel machine with O(N) processors.
